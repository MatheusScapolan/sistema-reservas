Logo INTELI

Para usar a logo real do INTELI, você precisa:

  1. Salvar a imagem PNG da logo do INTELI que você enviou
  2. Colocar o arquivo na pasta public/images/inteli-logo.png
  3. A tela de login já está configurada para usar essa imagem

Instruções:

1. Clique com o botão direito na imagem da logo do INTELI que você enviou

2. Selecione "Salvar imagem como..."

3. Salve como inteli-logo.png na pasta public/images/

Caminho do arquivo:

public/images/inteli-logo.png

Código já atualizado:

O código da tela de login já foi atualizado para usar a imagem real:

<img src="/images/inteli-logo.png" alt="INTELI Logo" class="inteli-logo">

CSS aplicado:

.inteli-logo {
    width: 150px;
    height: auto;
    filter: drop-shadow(0 4px 8px rgba(0, 0, 0, 0.1));
}
